# VoteSecure

> Secure, transparent, blockchain-inspired electronic voting platform  

## Live Demo  
[VoteSecure Live](https://vote-secure-alpha.vercel.app/)  

## Features  
- Real-time election monitoring  
- Blockchain-inspired vote security  
- Multi-position elections support  
- Role-based authentication (admin / voter)  
- Live analytics & results charts  
- Fraud detection alerts  
- Responsive admin & voter dashboard  

## Tech Stack  
React · TypeScript · Vite · Tailwind CSS · Supabase · Framer Motion · React Router · Recharts
